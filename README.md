# アニメ公式サイト

モダンで美しいデザインのアニメ公式サイトです。ReactとTailwindCSSを使用して構築されています。

## 特徴

- 🎨 美しいグラデーションデザイン
- 📱 完全レスポンシブ対応
- ✨ スムーズなアニメーション
- 🎭 キャラクター紹介
- 📺 エピソード情報
- 📰 最新ニュース
- 🌟 Framer Motionによる動きのあるUI

## 使用技術

- React 18
- TailwindCSS
- Framer Motion
- Lucide React Icons

## セットアップ

1. 依存関係をインストール：
```bash
npm install
```

2. 開発サーバーを起動：
```bash
npm start
```

3. ブラウザで `http://localhost:3000` を開きます

## ビルド

本番用にビルドする場合：

```bash
npm run build
```

## カスタマイズ

- カラーテーマ：`tailwind.config.js`で変更可能
- フォント：Noto Sans JPを使用
- アニメーション：Framer Motionで制御

## セクション構成

1. **ヒーローセクション** - インパクトのあるトップビュー
2. **ストーリー** - 作品のあらすじ紹介
3. **キャラクター** - 登場人物のプロフィール
4. **エピソード** - 各回の情報
5. **ニュース** - 最新情報
6. **フッター** - リンクと情報
